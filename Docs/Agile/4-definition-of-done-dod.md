# Definition of Done (DoD)

Un item se marca Done solo si:
- Cumple criterios de aceptacion.
- Respeta reglas de arquitectura y seguridad multitenant.
- Actualiza docs relacionadas (`-kb`, `-cc`, `-fs`) cuando aplique.
- Evidencia de validacion registrada (review, checklist, pruebas si aplica).
- Estado actualizado en backlog y sprint backlog.
- Mantiene independencia por app y degradacion graciosa si hay dependencias suaves.
